Server operators

On this site you can add your servers to participate in the NTP Pool system and review monitoring data and more. You can read more about joining the NTP Pool as a server operator (or continue to login).

Use the NTP Pool time service as a software or hardware vendor

If you distribute devices or software using the NTP Pool, you must sign up for "Vendor DNS Zone" (a set of unique hostnames for your organization).

Using the NTP Pool time service as an end-user

If you are an end-user not distributing software or devices to other users or organizations, you don't need an account to setup NTP.
